# 设计系统升级文档 v2.0.1

## 🎯 升级目标

本次设计升级专注于**护眼体验**和**信息可读性**，通过科学的配色方案和清晰的数据层次，为车险业务团队提供更专业、更舒适的数据分析体验。

## 🎨 新配色系统

### 核心配色

| 用途 | 颜色 | 色值 | 设计说明 |
|-----|-----|------|---------|
| D (最新周) | 主蓝色 | #5B8DEF | 柔和蓝，专业感，主数据突出 |
| D-7 (上周) | 次灰色 | #8B95A5 | 中性灰，辅助信息，不抢夺注意力 |
| D-14 (前周) | 浅灰色 | #C5CAD3 | 背景数据，最低视觉权重 |
| ↑ 上升 | 成功绿 | #52C41A | 正向鼓励，数据增长标识 |
| ↓ 下降 | 警示红 | #F5222D | 警示提醒，数据下滑标识 |
| — 持平 | 中性灰 | #8B95A5 | 无变化，保持稳定状态 |

### 设计原理

**护眼优先**
- 采用低饱和度色彩，减少视觉疲劳
- 科学的颜色对比度，确保长时间使用舒适
- 避免高亮度纯色，保护用户视力健康

**信息编码**
- 颜色承载业务含义：绿色=增长，红色=下滑，灰色=稳定
- 视觉权重分层：主数据>对比数据>背景数据
- 趋势符号强化：箭头方向+颜色变化双重编码

**专业克制**
- 参考Ant Design、Material Design企业级设计规范
- 保持界面简洁，避免过度装饰
- 统一视觉语言，提升品牌专业度

## 📊 标签格式升级

### 旧格式
```
2025-11-05: 790.7万
```

### 新格式
```
D (11-05): 791万 ↑ 20%
D-7 (10-29): 657万 ↓ 16%
D-14 (10-22): 781万 — 0%
```

### 格式规范

**日期显示**
- 简化年份：2025-11-05 → 11-05
- 统一格式：MM-DD，两位数月份和日期
- 时间流向：D-14 → D-7 → D，从左到右时间递进

**数值处理**
- 智能取整：790.7万 → 791万
- 单位统一：万为单位，避免小数干扰
- 精度控制：根据数值大小自动调整显示精度

**趋势计算**
- 基准对比：D vs D-7 计算增长率
- 符号标识：↑(增长) ↓(下降) —(持平)
- 百分比显示：保留整数位，去除小数

## 🏗️ 实施规范

### CSS变量定义

```css
:root {
  /* 主数据色彩 */
  --color-primary: #5B8DEF;
  --color-primary-light: #7BA3F3;
  --color-primary-dark: #4B7DE5;
  
  /* 辅助色彩 */
  --color-secondary: #8B95A5;
  --color-secondary-light: #C5CAD3;
  --color-secondary-dark: #6B7585;
  
  /* 状态色彩 */
  --color-success: #52C41A;
  --color-warning: #FAAD14;
  --color-error: #F5222D;
  
  /* 趋势色彩 */
  --color-trend-up: #52C41A;
  --color-trend-down: #F5222D;
  --color-trend-flat: #8B95A5;
}
```

### 组件应用

**KPI卡片**
- 主数值：使用主蓝色 #5B8DEF
- 对比数据：上周 #8B95A5，前周 #C5CAD3
- 趋势箭头：根据变化方向使用对应颜色

**周对比图表**
- 本周数据：主蓝色 #5B8DEF
- 上周数据：次灰色 #8B95A5  
- 前周数据：浅灰色 #C5CAD3
- 背景网格：使用更浅的灰色确保可读性

**筛选标签**
- 已选项：主蓝色背景+白色文字
- 悬停状态：主蓝色边框+主蓝色文字
- 禁用状态：浅灰色背景+灰色文字
