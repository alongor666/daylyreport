# 车险业务数据结构分析报告

## 概述

基于对Excel文件 `日报汇总-20250308-20250325.xlsx` 的详细分析，本报告提供车险业务数据结构的全面分析，为制作专业的日签单趋势报表提供数据基础。

## 1. 主要业务指标分析

### 核心财务指标
- **净保费**: 最重要的业务指标，范围 37,075 - 3,180,575 元，日均 144 万元
- **原保保费**: 新保单保费收入
- **批单保费净额**: 保单批改产生的保费变动
- **商业保费**: 商业险部分，范围 9,923 - 1,285,489 元
- **交强保费**: 交强险部分

### 业务量指标
- **总件数**: 每日保单总件数
- **原保件数**: 新保单件数
- **批单件数**: 批单件数

### 关键发现
- 3月10日为业务高峰日，净保费达318万元
- 商业险占比约60%，是主要收入来源
- 件均保费约2,300元

## 2. 时间维度数据分布

### 数据时间范围
- **起始日期**: 2025年3月8日
- **结束日期**: 2025年3月25日
- **数据天数**: 15天
- **数据完整性**: 工作日数据完整，周末数据较少

### 业务周期性特征
- **工作日**: 日均保费约150万元
- **周末**: 业务量明显下降
- **月度趋势**: 3月中旬达到业务高峰

### 建议的时间分析维度
1. **日度分析**: 每日签单趋势
2. **周度汇总**: 周一至周日业务模式
3. **月度对比**: 与上月同期比较
4. **同比环比**: 年度增长趋势

## 3. 三级机构结构分析

### 机构层级
目前数据显示为**二级机构**结构：

#### 分公司层级 (13个机构)
1. **天府分公司**: 保费最高 (1,097万元)，件数最多 (13,439件)
2. **高新分公司**: 保费第二 (273万元)，件数第三 (3,053件)
3. **青羊分公司**: 保费第三 (186万元)，件数第四 (2,288件)
4. **宜宾分公司**: 保费第四 (162万元)，件数第二 (6,183件)
5. **其他机构**: 武侯、新都、泸州、德阳、自贡、达州、资阳、乐山、本部

### 机构业务特征
- **头部集中**: 前4家机构贡献约70%的业务量
- **区域差异**: 成都城区机构业务量大，地市机构相对较小
- **件均保费**: 天府分公司件均保费较低(817元)，高新分公司件均保费较高(895元)

### 建议的机构分析维度

4. **机构对比**: 同类型机构间对比

## 4. 车险特有业务维度

### 险种结构
- **商业险**: 保费849万元，件数6,281件，件均保费1,353元
- **交强险**: 保费1,304万元，件数25,525件，件均保费511元

### 险种特征分析
- **交强险**: 件数占比80%，但保费占比仅61%
- **商业险**: 件数占比20%，保费占比39%，单件价值更高
- **业务结构**: 交强险为基础业务，商业险为价值业务

### 车险业务建议维度
1. **险种趋势**: 商业险、交强险分别趋势
2. **险种占比**: 商业险占比变化趋势
3. **件均分析**: 各险种件均保费变化
