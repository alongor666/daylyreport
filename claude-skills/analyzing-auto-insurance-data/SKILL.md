# 车险业务数据分析 Skill

## 核心功能

本Skill专门用于分析车险签单清单数据，提供全面的业务洞察和预警监控功能。

### 主要能力
- 📊 **Excel数据处理**: 读取、清洗、分析车险业务Excel文件
- 📈 **统计报告生成**: 自动生成多维度业务分析报告  
- 🏢 **机构绩效分析**: 三级机构、四级机构业务表现评估
- 👥 **客户结构分析**: 客户类别、续保情况、险别组合深度分析
- ⚠️ **业务预警监控**: 自动识别业务异常和波动
- 📅 **时间序列分析**: 支持按日、周、月的业务趋势分析

## 使用场景

### 1. 日常业务分析
```
请分析最近三周的车险业务数据，重点关注：
- 每日保费变化趋势
- 各机构业务表现对比
- 客户结构变化情况
```

### 2. 异常监控预警
```
运行业务预警监控，检查：
- 日保费波动是否超过10%阈值
- 周末业务是否存在异常激增
- 机构集中度是否过高
```

### 3. 专项深度分析
```
对摩托车客户群体进行深度分析：
- 保费贡献占比变化
- 续保率表现
- 险别组合偏好
```

## 数据文件要求

### 输入文件格式
- **Excel文件**: `.xlsx` 或 `.xls` 格式
- **CSV文件**: `.csv` 格式，UTF-8编码
- **JSON配置**: 业务员与机构对应关系

### 核心数据字段
```csv
投保确认时间, 报告日期, 三级机构, 四级机构, 业务员, 
客户类别, 险别组合, 续保情况, 总保费, 手续费, 签单保额
```

## 分析维度

### 时间维度
- 📅 **日度分析**: 每日业务量、保费收入、趋势变化
- 📊 **周度对比**: 三周同星期对比分析
- 🎯 **时段分析**: 工作日vs周末业务特征

### 机构维度  
- 🏢 **三级机构**: 按照业务员对应关系确定真实三级机构
- 🏪 **四级机构**: 支公司层面业务分析
- 📍 **地域分析**: 成都市vs其他城市对比

### 客户维度
- 👤 **客户类别**: 9大类客户群体分析
- 🔄 **续保情况**: 转保、续保、新保三大类别
- 🛡️ **险别组合**: 5种主要险别组合分析

## 业务规则

### 数据清洗规则
1. **负值保费**: 保留分析（批改退保导致）
2. **零手续费**: 正常现象，无需处理
3. **保额负值**: 异常值，建议忽略

### 预警阈值
- **日保费波动**: ±10%为异常阈值
- **周末增长**: 超过10倍为异常
- **机构集中度**: 单一机构占比超过40%为高风险

### 核心洞察
- **高价值客户**: 非营业个人客车（53.8%占比）
- **关注群体**: 摩托车（24.8%）、非营业货车（5.6%）
- **险别结构**: 单交业务48.1%，商业险提升空间大

## 执行流程

### 标准分析流程
1. **数据加载**: 读取Excel/CSV文件
2. **数据清洗**: 处理异常值、缺失值
3. **基础统计**: 计算核心业务指标
4. **维度分析**: 按时间、机构、客户维度分析
5. **异常检测**: 运行预警规则
6. **报告生成**: 输出分析结果和建议

### 预警监控流程
1. **数据准备**: 加载最新业务数据
2. **规则执行**: 运行6大监控模块
3. **结果分类**: 按高/中/低优先级分类
4. **报告输出**: 生成监控报告

## 最佳实践

### 数据质量
1. 确保投保确认时间字段完整
2. 机构名称使用标准命名
3. 客户类别分类准确

### 分析建议  
1. 重点关注日保费波动超过10%的异常
2. 周一波动需特别关注（209万→318万→274万）
3. 周末业务激增需验证是否为调休因素

### 业务洞察
1. 天府机构占比42%，需关注集中度风险
2. 个人客户赔付率更低，企业客户风险更高
3. 48%单交业务存在商业险渗透机会
