# 📖 跨操作系统主题系统 - 使用指南

## 🎯 快速开始

### 1. 访问系统
打开浏览器，访问：
```
http://localhost:8088
```

### 2. 主题切换
- **点击切换**: 右上角的主题切换按钮
- **快捷键操作**:
  - `Alt + T`: 切换主题
  - `Alt + E`: 护眼模式
  - `Alt + D`: 暗黑模式

### 3. 系统偏好
- 自动检测系统深色/浅色偏好
- 可手动设置跟随系统或固定模式
- 支持减少动画的护眼设置

## 🎨 主题模式详解

### 👁️ 护眼模式
**适合场景**: 白天办公、长时间阅读、视力保护

**使用建议**:

### 🌙 暗黑模式
**适合场景**: 夜间使用、低光环境、演示汇报

**视觉特征**:
- 🌑 纯黑背景减少亮度

- 🎯 专业的夜间配色

**使用建议**:
- 夜间或低光环境使用
- 演示汇报时增强视觉效果

## ⌨️ 快捷键操作

### 主题切换快捷键
| 快捷键 | 功能 | 适用场景 |
|--------|------|----------|
| `Alt + T` | 切换主题模式 | 快速在护眼/暗黑间切换 |
| `Alt + E` | 护眼模式 | 立即切换到护眼模式 |
| `Alt + D` | 暗黑模式 | 立即切换到暗黑模式 |

### 使用技巧
1. **单手操作**: 左手按住Alt，右手按T/E/D
2. **盲操作**: 无需看屏幕即可完成切换
3. **快速响应**: 切换延迟<200ms
