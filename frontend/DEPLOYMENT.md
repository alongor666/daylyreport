## 🔧 部署配置

### 生产环境配置
```typescript
// 推荐配置
const themeOptions = {
  defaultMode: 'eye-care',        // 默认护眼模式
  respectSystemPreference: true,  // 跟随系统
  enableTransitions: true,        // 平滑过渡
  enableCache: true,              // 启用缓存
  performanceThreshold: 50        // 性能阈值
}
```

### CDN部署
```html
<!-- 引入主题系统 -->
<link rel="stylesheet" href="/assets/index-DGjLZjJN.css">
<script type="module" src="/assets/index-OpvJ2XgY.js"></script>
```

### 服务器配置
```nginx
# Nginx配置示例
location / {
    root /path/to/dist;
    try_files $uri $uri/ /index.html;
    
    # 缓存静态资源
    location ~* \.(js|css|png|jpg|svg)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }
}
```
