# 车险签单数据分析平台 v2.0 前端

本目录包含使用 Vue 3 + Vite + Pinia 构建的前端代码。开发流程、组件规范与设计要求请参见仓库根目录的 `CLAUDE.md`、`docs/README_FOR_DEVELOPERS.md` 与 `docs/DESIGN_SYSTEM.md`。

## 开发命令

```bash
npm install
npm run dev
npm run build
npm run lint
```

开发服务器默认运行在 `http://localhost:3000`，并通过 Vite 代理将 `/api` 请求转发到 `http://localhost:5001`。
